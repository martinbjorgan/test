Provisioning infrastructure on to AWS using Terraform
EVRY!

The following resource have been provisioned using Terraform

  • 1 VPC (Virtual Private Cloud)
  • 1 Internet gateway
  • 1 AWS route - Gives Internet access to the VPC
  • 1 Subnet
  • 1 ELB (Elastic Load Balander)
  • 2 Security groups
    • One for the ELB (Elastic Load Balancer) --> Tied to the Web server interface
    • One default --> Gives SSH,HTTP and MySQL access to the interfaces
  • 2 Ubuntu Linux instances
    • 1 Web server with Apache installed --> Via script
    • 1 Database server with MySQL installed and database + tables created --> Via script
